# 🤖 Auto Link Telegram Bot

A powerful Telegram bot that automatically sends incremented message links every 15 seconds.

---

## 🚀 Features

- 🔁 Auto link sender (72 → 73 → 74…)
- ⏱️ 15 sec auto message system
- 💾 MongoDB database support
- 👥 Multi-user system
- 🛑 Start / Stop commands
- 🎛️ Inline buttons (Help, About, etc.)
- 📢 Log channel tracking
- ☁️ Deploy on Render

---

## 🧠 Commands

| Command | Description |
|--------|------------|
| /start | Start auto sending |
| /stop  | Stop sending |
